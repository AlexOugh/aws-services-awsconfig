
# AWSConfig

AWS Lambda functions to manage the AWSConfig Service

![aws-services][aws-services-image]

## How To Setup

    $ AWS CodePipeline, 'aws-services-awsconfig'


## How To Update Lambda Function Codes

    $ run_update_codes


## How To Test Lambda Functions

    $ cd tests
    $ node test_xxx.js
